# Getting Started with RFM Analysis

## What This Project Does

This project analyzes 99,000+ customers from a Brazilian e-commerce platform and segments them using RFM analysis to:

✅ Identify your best customers (Champions)  
✅ Find at-risk customers before they leave  
✅ Personalize marketing strategies  
✅ Maximize revenue by segment  
✅ Make data-driven business decisions  

## Files in This Project

### 📊 Data Files (CSV)
- **rfm_all_customers.csv** - Complete customer data with segments (99K+ rows)
- **rfm_champions.csv** - Your best 16K customers
- **rfm_at_risk.csv** - 42K customers at risk of leaving
- **rfm_segment_summary.csv** - High-level overview
- **rfm_geographic.csv** - State-by-state analysis

### 📁 Code Files
- **sql/** - SQL queries for analysis
- **python_scripts/** - Visualization code
- **visualizations/** - 8 professional charts

### 📖 Documentation
- **DATA_DICTIONARY.md** - Column explanations
- **SEGMENT_GUIDE.md** - Understanding segments
- **MARKETING_STRATEGIES.md** - Action plans per segment
